## Setup Instructions

1. Clone the Repository:
```
git clone https://github.com/your-username/interactive-form.git
cd interactive-form
```

2. Install Dependencies:
```
npm install
```


3. Run the App Locally:
```
npm start
```

5. Create Build Folder:
```
npm run build
```


6. Deploy on Netlify:

Link the repository to your Netlify account.
